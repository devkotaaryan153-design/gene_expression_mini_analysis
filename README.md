## Gene Expression Mini Analysis

This project explores gene expression differences between healthy and diseased conditions using pandas.

### Steps performed:
- Cleaned dataset columns
- Grouped gene expression by condition
- Created quantile-based risk categories
- Built pivot table for comparison
- Calculated expression differences between conditions

### Tools used:
- Python
- Pandas

### Key insight:
Some genes show higher expression in diseased conditions, indicated by negative differences in the pivot table.
